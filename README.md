# BIGDATA_ANALYTICS_exercicios_aula

Este repositório contém exercícios e scripts desenvolvidos para fins acadêmicos no contexto do **Mestrado em Computação Aplicada** do **Instituto de Pesquisas Tecnológicas (IPT)**.

## Conteúdo

- Scripts em R (`.R`) com atividades e exercícios das aulas
- Aplicações interativas (por exemplo, Shiny)
- Documentos de apoio (quando aplicável)

## Observações

- O material aqui disponibilizado tem finalidade **exclusivamente acadêmica**.
- Caso existam arquivos de dados utilizados nos exercícios, eles podem não estar incluídos no repositório.
